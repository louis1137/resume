본인 소개

Javascript기반의 10년차 프론트엔드 개발자입니다.
JS를 활용한 웹페이지 작업을 많이 하였으며, 작업시 작업의 효율성 높이는 것을 중요하게 생각합니다.
현재 관심있는 기술로는 3D 핸들링툴인 webGL과 생성형AI(stable diffusion, Midjourney, openAi)에 관심을 가지고 공부하고 있습니다.

업무 철학

  • 소스 코드의 재활용 소스 코드는 최대한 재활용, 같은 작업 반복을 지양
  • 기능 구현의 모듈화 작은 기능을 구현하더라도 해당 기능을 모듈화하고, 파라미터를 적극 사용하여 전역 객체의 사용 최소화
  • 레거시의 최소화 모듈 형태로 함수를사용함으로써, 해당 기능이 삭제되는 시점에 레거시도 함께 제거
  • 가이드 제시 기능을 구현하고, 해당 기능을 배포 시 라이브러리 형태로 가이드를 작성하여 같이 작업하는 작업자에게 공유
  • 객체의 기능 확장 javascript 내장 객체 (Array, Object, String등)에 필요 기능들을 prototype을 활용하여 기능을 추가
  • JSON의 활용 관리가 필요한 데이터들은 .json파일로 따로 분리해 용이한 관리 지향

현재 업무

  • 팀원 관리 및 프로젝트 리딩 - 프로젝트 진행 팀원 대상 교육 및 프로젝트 스케쥴 관리
  • restAPI를 사용한 데이터 핸들링 - 동기/비동기화의 원리를 이해하고, 해당 웹 사이트를 동적으로 동작하도록 작업
  • 프로젝트에서 필요한 함수 제작, 배포 및 가이드 - 작업자들이 필요한 함수를 모듈 형태로 만들어 제공하고, 플로그인 형태로 사용 가능하도록 작업 및 가이드 배포
  • javascript를 사용한 micro front-end 기술을 활용하여 다수의 사이트에 하나의 컴포넌트를 제공 - 많은 홈페이지의 유지 보수를 위해 micro front-end 기술 적용 - 다양한 섹션 선 작업 후, 해당 영역에 필요한 코드를 파싱하도록 작업 진행 (react의 component 기능) - SPA사이트 제작 경험
  • react를 활용한 사내 직원들 전용 dashboard 유지 보수
